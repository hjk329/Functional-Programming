# Array.prototype.reduce()

```js
const nums = [1,2,3,4,5];

let total = 0;
for (const n of nums){
  total = total + n;
}

console.log(total); // 15
```

특정 값을 순회하면서 하나의 값으로 누적하기 위해서 사용한다.  

## 추상화하기

```js
const reduce = (f, acc, iter) => {
  for (const a of iter){
  acc = f(acc,a);
  }
  return acc;
}

const add = (a,b) => a + b;

console.log(reduce(add, 0, [1,2,3,4,5]));
```
